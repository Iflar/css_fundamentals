Font

In CSS, there are two font family names/types:

  • generic-family: a group of font families with sillimar look (ex: Serif)
  • font family: a specific font family (ex: "Times New Roman"")

The font-family property should hold several font names as a fallback in case the browser dosn't support the first font that is chosen.
Always start with the font you want, and end with a generic family. Also, if the font has more than one word, it must be in quotation marks - "Times New Roman"

Here is a list of the CSS font properties:

font:
sets all font properties in one decliration
font-family
Specifies font family for text
font-size
Specifies font size
font-style
font style for text
font-weight
specifies weight of font
